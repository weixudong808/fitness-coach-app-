## 🚀 启动步骤（3步搞定）

### 1. 打开终端
- 按 `Command + 空格`
- 输入 "Terminal"
- 回车

### 2. 复制粘贴这段命令（一次性搞定）
```bash
cd /Users/quhongfei/Documents/code/闲聊/fitness-coach-app && lsof -ti:5173 | xargs kill -9 2>/dev/null; pnpm dev
```

**说明：** 这一条命令会自动：
- 进入项目目录
- 停止旧的服务器（如果有）
- 启动新的服务器

### 3. 等待启动成功
看到这个提示说明成功：
```
VITE v7.3.1  ready in xxx ms
➜  Local:   http://localhost:5173/
```

然后打开浏览器访问：**http://localhost:5173/**

## 🔧 常见问题

### Q: 提示 "vite: command not found" 怎么办？
**A:** 依赖没装好。执行：
```bash
cd /Users/quhongfei/Documents/code/闲聊/fitness-coach-app
pnpm install
pnpm dev
```

### Q: 提示 "Port 5173 is in use" 怎么办？
**A:** 旧服务器还在运行。执行：
```bash
lsof -ti:5173 | xargs kill -9
pnpm dev
```

### Q: 网页打不开显示"连接被拒绝"？
**A:** 服务器没运行。回到终端，重新执行启动命令。

### Q: 提示 "pnpm: command not found" 怎么办？
**A:** pnpm 没安装。执行：
```bash
npm install -g pnpm
```
