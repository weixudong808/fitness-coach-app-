## 📊 数据库结构

### 核心表
1. **users** - 用户表（Supabase Auth）
2. **user_roles** - 用户角色表（coach/member）
3. **members** - 会员信息表
4. **training_templates** - 训练模板表
   - `is_template` - 区分模板和专属计划
   - `member_id` - 专属计划关联的会员
5. **training_sessions** - 训练课次表
6. **session_exercises** - 课次动作表
7. **member_plans** - 会员训练计划表
8. **member_session_records** - 会员课次记录表 ⭐
9. **member_exercise_records** - 会员动作记录表 ⭐

## 🐛 已知问题和解决方案

### 问题1：npm 无法正确安装 vite
**原因**：Node v25.6.1 版本过新，npm 存在兼容性问题

**解决方案**：
1. 使用 nvm 切换到 Node 20.x LTS
2. 使用 pnpm 替代 npm

### 问题2：开发服务器无法启动
**检查清单**：
1. Node 版本是否为 20.x：`node --version`
2. 依赖是否正确安装：`ls node_modules/.bin/vite`
3. Supabase 配置是否正确：检查 `.env` 文件
4. 端口是否被占用：`lsof -ti:5173`
