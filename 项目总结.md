## 🎯 项目目标
为健身私人教练打造一个会员管理系统，包含教练端和会员端两个端口：
- **教练端**：管理会员信息、创建训练计划、查看会员进度
- **会员端**：查看训练计划、记录训练数据、追踪自己的进步

## 🐛 常见问题

### Q1: 启动项目时报错"Cannot find package 'vite'"
**解决方法**：
```bash
npm install vite@latest --save-dev
npm run dev
```

### Q2: 登录时提示"Invalid API key"
**解决方法**：
1. 检查 `.env` 文件是否存在
2. 确认API密钥是否完整
3. 重启开发服务器（Control+C 然后 npm run dev）

### Q3: 数据库操作失败
**解决方法**：
1. 检查Supabase项目是否正常运行
2. 确认行级安全策略是否正确设置
3. 查看浏览器控制台的错误信息

### Q4: 如何添加新的数据库表？
**解决方法**：
1. 登录Supabase
2. 进入SQL Editor
3. 编写CREATE TABLE语句
4. 执行SQL
5. 设置相应的RLS策略
