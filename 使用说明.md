## 下一步操作

### 第一步：配置Supabase（必须完成）

在运行项目之前，你需要：

1. **注册Supabase账号**
   - 访问 https://supabase.com
   - 注册并创建新项目

2. **创建数据库表**
   - 按照 `SUPABASE_SETUP.md` 文档中的SQL脚本创建所有表

3. **配置环境变量**
   ```bash
   cp .env.example .env
   ```
   然后编辑 `.env` 文件，填入你的Supabase配置

4. **创建测试用户**
   - 在Supabase中创建一个教练账号
   - 在 `user_roles` 表中设置角色为 'coach'

### 第二步：启动项目

由于npm scripts可能存在路径问题，推荐使用以下方式启动：

```bash
# 方式1：使用npx（推荐）
npx vite

# 方式2：使用启动脚本
./start-dev.sh

# 方式3：如果npm run dev能工作
npm run dev
```

启动后访问：http://localhost:5173

### 第三步：登录测试

使用你在Supabase中创建的教练账号登录，你应该能看到：
- 会员管理页面
- 可以添加、编辑、删除会员

## 常见问题

### Q: 启动项目时报错？
A:
1. 确保已安装Node.js 18+
2. 确保已运行 `npm install`
3. 尝试使用 `npx vite` 启动
4. 检查 `.env` 文件是否正确配置

### Q: 登录失败？
A:
1. 确认Supabase项目已创建
2. 确认数据库表已创建
3. 确认用户已在Supabase中创建
4. 确认用户角色已在 `user_roles` 表中设置

### Q: 会员列表为空？
A: 这是正常的，因为还没有添加会员。点击"添加会员"按钮开始添加。

### Q: 如何添加更多教练账号？
A:
1. 在Supabase Authentication中创建新用户
2. 在 `user_roles` 表中添加记录，设置 `role` 为 'coach'
